# ltsm

The files in this directory are copied **as-is** from the LINE Messenger Chrome extension (`ophjlpahpchlmihnnnihgmmeilfjmjjc`):

- `ltsm.wasm` — WebAssembly crypto module responsible for HMAC signing and key derivation.
- `ltsmSandbox.js` — JavaScript sandbox that loads and wraps the WASM module.

Do not edit these files. They are binary/generated artifacts owned by LINE Corporation.
